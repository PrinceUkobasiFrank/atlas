# Research Companion

A lightweight research companion designed for independent researchers, designers, engineers, and students.

> "A beautiful desk where ideas are collected."

https://useatlas.netlify.app/

## Features

- **Contextual Greeting** — Time-aware greeting based on local time
- **Today's Focus** — Editable daily focus, persisted locally
- **Quick Notes** — Create, edit, delete, and search notes with timestamps
- **Research Questions** — Track open questions with status (Open / Investigating / Answered)
- **Sources** — Store and access research references with URLs
- **Instant Search** — Filter across notes, questions, and sources in real time
- **Dark / Light Mode** — Theme preference persisted locally
- **Export** — Export all data as JSON with a single click

## Technology

- TypeScript
- Vite
- HTML
- CSS

No frameworks. No unnecessary dependencies.

## Installation

